Прехвърляне от популярни български блог-хостинг услуги в WordPress блогове.

== Description ==
Този плъгин (приставка, разширение) е направен, за да направи по-лесно прехвърлянето на блогове от различни популярни български блог-хостинг услуги в WordPress блогове (в момента само <a href="http://blog.bg">Blog.bg</a> и <a href="http://log.bg">Log.bg</a>).

== Installation ==
1. Изтеглете инсталационния файл и го разархивирайте
1. Качете разирхивираното съдържание в папката с приставките на вашия WordPress блог (казва се wp-content/plugins/)
1. Активирайте приставката (плъгина, разширението) в административния панел на WordPress (това е на Plugins страницата)
1. Активирайте модула, който ви е необходим от Imports страницата в административния панел на WordPress
1. Ами това е ;)
